Sleep Detection Software

Introduction

This project is a sleep detection software based on MediaPipe and YOLOv8. It aims to analyze and monitor sleep patterns, including sleep state detection and posture recognition. The software is still under development, and more features will be implemented in future updates.

Features

Implemented Features:

Sleep state detection

Sleep posture recognition

Planned Features:

Development Plan

This project is being actively developed, and new features will be gradually integrated.

Technologies Used

MediaPipe: Used for pose estimation and keypoint tracking.

YOLOv8: Used for object detection (e.g., detecting the presence of a blanket).

Python: The primary programming language for development.

OpenCV: Used for image processing and visualization.

NumPy/Pandas: Used for data analysis and report generation.
